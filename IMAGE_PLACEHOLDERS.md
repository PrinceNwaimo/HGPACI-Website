# Image Placeholders

The following image placeholders are used throughout the website and should be replaced with actual images:

## Homepage (HomePage.tsx)
- `hero-worship-image.jpg` - Hero section background: Diverse congregation worship service with raised hands
- `community-fellowship-image.jpg` - Community section: Church fellowship and people connecting

## About Page (AboutPage.tsx)
- `church-worship-image.jpg` - Worship service with congregation

## Service Times Page (ServiceTimesPage.tsx)
- `church-building-image.jpg` - Exterior view of church building

## Ministries Page (MinistriesPage.tsx)
- `worship-ministry-image.jpg` - Worship team leading praise
- `prayer-ministry-image.jpg` - Prayer gathering
- `healing-ministry-image.jpg` - Healing and deliverance ministry
- `youth-ministry-image.jpg` - Youth group activities
- `outreach-ministry-image.jpg` - Community outreach and service

## Events Page (EventsPage.tsx)
- `conference-event-image.jpg` - Conference or special event gathering

## Recommended Image Specifications
- Format: JPG or PNG
- Minimum resolution: 1920x1080 for hero images, 800x600 for other images
- Aspect ratio: 16:9 for hero images, 4:3 or 16:9 for content images
- File size: Optimized for web (under 500KB per image)

## How to Replace
1. Add your images to the `/public/images/` directory
2. Update the image src attributes in the respective page files
3. Ensure alt text accurately describes the image content
